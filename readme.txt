Custom weight-based shipping method for WooCommerce with configurable base rates and per-kg charges.

== Description ==

DX Shipping for WooCommerce provides a flexible weight-based shipping calculation method that's perfect for businesses needing custom shipping rates based on package weight.

= Features =

* **Weight-based calculations** - Set a base rate for orders up to a weight threshold
* **Excess weight charges** - Add per-kg charges for weight exceeding your threshold
* **Flexible configuration** - Customize rates for different shipping zones
* **Free shipping option** - Offer free shipping for orders above a certain amount
* **Min/Max constraints** - Set minimum and maximum shipping costs
* **Debug mode** - Log calculations for troubleshooting
* **Multi-zone support** - Configure different rates for different regions
* **Tax support** - Compatible with WooCommerce tax settings

= Default Configuration =

* Base rate: £8.00 for orders up to 20kg
* Excess rate: £0.40 per kg over 20kg
* Fully customizable through WooCommerce shipping zones

= Use Cases =

* E-commerce stores with weight-based shipping
* Businesses shipping heavy or bulky items
* UK-based retailers using DX delivery services
* Any store needing custom weight-based calculations

== Installation ==

= Automatic Installation =

1. Go to Plugins > Add New in your WordPress admin
2. Search for "DX Shipping for WooCommerce"
3. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin zip file
2. Go to Plugins > Add New > Upload Plugin
3. Choose the downloaded file and click "Install Now"
4. Activate the plugin

= Configuration =

1. Go to WooCommerce > Settings > Shipping
2. Add or edit a shipping zone
3. Click "Add shipping method"
4. Select "DX Shipping" from the dropdown
5. Configure your rates and save

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =

Yes, WooCommerce must be installed and activated for this plugin to work.

= Can I set different rates for different zones? =

Yes, you can configure different rates for each shipping zone in WooCommerce.

= What weight units are supported? =

The plugin supports all weight units configured in WooCommerce (kg, g, lbs, oz).

= Can I offer free shipping? =

Yes, you can set a free shipping threshold based on order total.

= Is tax calculated on shipping? =

Yes, the plugin respects your WooCommerce tax settings for shipping.

= Can I test my shipping calculations? =

Yes, the plugin includes a calculator tool in the admin area for testing rates.
